A variable is a container for storing data and values.
A variable has four keys which are the variable, variable name, assignment operator and the value

A data type is what a variable operates on. Data type id the type of the value of a variable.

There are six types of data types (that i know)

  1. String :A string is any character that is within a quote '' can be single or double quote.
  2. Number :A number is any number assigned to a variable that is not inside a quote. Numbers can be positive, negative, real, floating, double etc.
  3. Boolean :A boolean is a type of data that can either be true or fasle, yes or no.
  4. Undefined :Undefined is used when the variable does not have value.
  5. Null :A null variable is a type of variable that has null as the value. Javascript classify null as an object.
  6. Object :An object is a container that can hold more than one value.
